# SA-MP Poker
SA-MP Poker is a complete Texas Hold 'em Poker system for the San Andreas Multiplayer modification Grand Theft Auto: San Andreas. The system is built specifically for [Los Santos Roleplay](https://ls-rp.com).

## Languages Used
- PAWN - Game Scripting
- C++ - Hand Evaluator
